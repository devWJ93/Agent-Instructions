# KakaotalkChat (PlayMCP)

> 카카오톡 "나에게 보내기"(메모) 기능을 제공하는 PlayMCP 하위 서버.

## 📌 용도
카카오톡의 "나와의 채팅(메모)"으로 메시지를 전송한다. Claude가 작업 결과·알림·메모를 사용자 본인의 카카오톡으로 보낼 수 있게 한다.

## 🎯 트리거 조건 / 사용 맥락
- "카카오톡으로 보내줘 / 나한테 메모 남겨줘 / 결과를 카톡으로 알려줘" 같은 요청
- 작업 완료 알림, 요약, 리마인더를 본인 카카오톡으로 전달할 때

## 🛠️ 제공 도구 목록

| 도구명 | 기능 | 주요 파라미터 |
|--------|------|----------------|
| MemoChat | 카카오톡 "나에게 보내기"(나와의 채팅)로 메시지 전송 | `text`(보낼 메시지 본문) 등 메시지 내용 |

> 본인에게 보내는 메모 채널이므로 타인에게 메시지를 보내는 기능은 아니다.

## 💻 사용 예시

**예시 1 — 작업 완료 알림**
- `MemoChat`(text: "빌드/배포 완료. 에러 0건.") → 사용자 카카오톡 나와의 채팅에 도착

**예시 2 — 요약 전달**
- 긴 조사 결과를 정리한 뒤 `MemoChat`으로 핵심 3줄 요약 전송

## 📦 출처 / 설치 상태
- 제공: **claude.ai 연동 / PlayMCP 하위 서버** (카카오 공식)
- 활성화: PlayMCP 묶음으로 연결됨 (`claudeAiMcpEverConnected: ["claude.ai PlayMCP"]`)
- 연결 방식: claude.ai 원격 MCP (클라우드 호스팅). 카카오 계정/카카오톡 인증 기반
- 세션 내 도구 식별자: `mcp__claude_ai_PlayMCP__KakaotalkChat-MemoChat`
